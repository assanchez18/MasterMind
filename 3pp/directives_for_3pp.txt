https://github.com/tronkko/dirent

To make dirent.h available for all C/C++ programs, simply copy the include/dirent.h file to 
the system include directory. System include directory contains header files such as assert.h 
and windows.h. In Visual Studio 2008, for example, the system include may be found at 
C:\Program Files\Microsoft Visual Studio 9.0\VC\include

search for assert.h in microsoft visual studio directory
